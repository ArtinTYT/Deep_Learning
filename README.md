# 李沐_手动深度学习

## 项目简介

本项目包含李沐老师的《动手学深度学习》一书的代码实现，对代码加上了详细的注释，方便学习。

## 内容来源

内容是根据李沐老师的《手动深度学习》[https://zh.d2l.ai/chapter_preface/index.html](https://zh.d2l.ai/chapter_preface/index.html) 这本书的代码，对代码加上了详细的注释，方便学习。

## 技术栈

- **框架**: PyTorch
- **语言**: Python
- **格式**: Jupyter Notebook

## 学习建议

1. 按照目录顺序学习，从基础知识开始
